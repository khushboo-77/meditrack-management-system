# JVM Overview

## JVM Components
- Class Loader
- Runtime Data Areas (Heap, Stack, Method Area)
- Execution Engine

## Heap vs Stack
- Objects are stored in Heap
- Method calls and local variables are stored in Stack

## JIT Compiler
- Improves performance by compiling frequently used bytecode

## Write Once Run Anywhere
- Java compiles to bytecode
- JVM executes bytecode on any OS
